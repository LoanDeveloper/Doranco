# Benchmarking IA

## Consignes générales

Un travail de recherche approfondi est demandé, listez les modèles utilisés, l'historique du sujet, les défis industriels que le domaine est susceptible de résoudre en rapport avec l'étude de cas que vous avez sélectionnée.

Les sources et flux identifiés doivent répondre à ces critères de fiabilité :

- L’auteur de la page est clairement identifié
- Des informations sur l’auteur sont disponibles et confirment ses compétences et l’absence d’intérêts personnels.
- L’analyse du contenu est valable (date de publication récente, sources de l'information indiquées, niveau de langue correct),
- L’information peut être confirmée par d’autres sites de confiance (dans la mesure du possible)

Expliquer les métriques utilisés pour tester et améliorer les modèles

Listez les modèles/services déjà disponibles, comment ont-ils été entraînés, quelle volumétrié de données ?

Expliquez les architectures utilisées, les fonction de perte, les techniques permettant de réduire l'overfitting etc.. Quelles méthodes récentes ont été utilisées ?

Quelles sont les limites de chaque modèle/solution en place ?

Quelles perspectives dans le futur?

Quels moyens sont nécessaires pour entraîner/faire l'inférence ?

Faites un choix de solutions dans votre synthèse (une solution gold, puis une solution de repli)

Identifier les freins potentiels (juridique, éthique, coût et
technique)

Votre livrable se présente sous la forme d'une présentation slide à faire devant la classe, les autres étudiants challengerons vos solutions choisies.

Basez vous sur des sources comme :

https://huggingface.co/
https://www.kaggle.com/


Pour votre étude de cas , réalisez un plan de mitigation des risques via cette structure :

| Critère     | Description                                |
| ----------- | ------------------------------------------ |
| Probabilité | Faible / Moyenne / Forte                   |
| Impact      | Mineur / Majeur / Critique                 |
| Nature      | Technique / Juridique / Financier / Humain |
| Périmètre   | Métier, SI, conformité, image              |

Imaginez au moins trois risques possibles, basés sur vos recherches.

Vous devez également réfléchir à l'identification des enjeux éthiques et légaux relatifs à la collecte des données personnelles.
